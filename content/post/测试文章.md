## 新建文章

```shell
hugo new post/文章名.md
```

## 撰写文章

使用 `Markdown` 语法：

# 第一级标题

## 第二级标题

**加粗**样式。

- 牛奶
- 香蕉
- 苹果

1. 热油
2. 下菜
3. 翻炒
4. 承盘

这里就是正文内容。

## Markdown 预览

点击 `Ctrl+Shift+P` 打开命令栏，选择【打开侧边预览】进行预览。

## 发布

记得删除文章前面的 `draft=true` 

### 使用 Git

点击 `VS Code` 的 `Git` 菜单，点击【更改】 的 `+` 号进行暂存。在消息填入提交消息，然后点击提交即可进行发布。
